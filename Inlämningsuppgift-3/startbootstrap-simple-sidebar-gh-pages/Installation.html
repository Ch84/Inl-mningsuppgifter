Installation


Välkommen till våran avdelning för installation!


Behöver du hjälp med att installera mjuk eller hårdvara?

Vi kan hjälpa till med det!

Det gäller oavsett om det gäller installation av operativsystem såsom t.ex:

  • Microsoft Windows
  • Unix/Linux
  • Ubuntu
  • Apple
Vi hjälper även till med förslag och installation av antivirus program. Vi har dom flesta märken inne såsom:

  • Norton
  • BullGuard
  • McAffee
  • TotalAV
  • Avast
  • F-Secure
Det är några utav våra märken som räknades upp här, men det finns såklart fler.

Vi hjälper även till med att installera mer avancerad mjukvara. Det kan t.ex. röra sig om:

  • MySQL Server
  • MS SQL Cluster
  • Windows Cluster
  • Hyper-V Cluster
Vi hjälper även till med ominstallation. Innebär att du bara lämnar in datorn så formaterar vi hårddisken och installerar om ditt operativsystem.

Om du behöver hjälp med att installera hårdvara är du också välkommen att vända dig till oss. Vi kan hjälpa dig att installera:

  • Skrivare
  • Webbkameror
  • Servrar
  • Processor
  • Hårddiskar (Både interna och externa
  • Moderkort
  • Scanner
  • Bildskärm
  • Tangentbord
  • Mus
Listan kan göras lång.

Vi har följande drop in tider:

  • 10:00 - 12:00
  • 13:00 - 15:00

Har du övriga frågor?


Du är välkommen att kontakta oss. Du hittar kontaktuppgifter till oss genom att klicka på Meny och sedan Kontakt. Du hittar även våra öppettider under kontakt.

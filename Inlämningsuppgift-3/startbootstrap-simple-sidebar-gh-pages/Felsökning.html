Felsökning


Välkommen till våran avdelning för felsökning!


Vi hjälper till att felsöka din dator oavsett problem.

Det kan kanske handla om att ditt operativ system inte startar upp korrekt, att datorn är långsam eller allmänt seg så du inte längre kan jobba i den prestanda du hade från början. Detta kan bero på att det finns spyware eller malware program, virus eller andra processer i bakgrunden som du inte ser som gör att datorn jobbar helt i onödan.

Vi hjälper självklart också till om det är någon form av hårdvaruproblem på din dator. Det kan röra sig om att datorn kanske inte vill starta igång alls när du trycker på start knappen. Kanske är det glapp i sladdarna? Moderkortet? Hårddisken? Eller kanske till och med processorn? Vi hittar felet garanterat.

Eller kan det kanske vara så att datorn inte hittar nätverket och det behövs en nätverks felsökning? Vi hjälper till med det också.

Vad det än är för fel så har vi tekniker på plats som gör en grundlig genomgång av din dator för att finna felet.

Allt detta till ett fast pris för bara 200 kr. Efter att våra tekniker har hittat felet kommer dom att kontakta dig på angivet telefonnummer för att diskutera vidare reparation samt pris förslag.

Vi har följande drop in tider:

  • 10:00 - 12:00
  • 13:00 - 15:00
